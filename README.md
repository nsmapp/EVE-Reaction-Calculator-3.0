## EVE Reaction Calculator 3.0  
Is a compact Android application designed to calculate reaction manufacturing in the game EVE Online. 

### Core libraries used in the app:
* Jetpack Compose 
* Ktor 
* SQLDelight 
* Koin (with anatations) 
* Jetpack Navigation 3 
* Coin
